## Gates run (every wave)

`tsc --noEmit` 0 ¬∑ vitest (final: 282 passed) ¬∑ `next lint` ¬∑ prettier ¬∑
`next build` ¬∑ hex/raw-font-size greps ¬∑ responsive probes 360/768/1280
(no doc-level horizontal scroll) ¬∑ Playwright behavioral probes against a
mock-mode dev server ¬∑ screenshot comparison against the handoff
`screenshots/*.png` at artboard widths.

**¬ß4a review substitution (Codex is not installed in the cloud
container ‚Äî see `docs/review-gate-cloud.md`):** every wave's staged diff
was reviewed by a freshly-spawned INDEPENDENT review agent (not the
authors), findings dispositioned by the lead, and all Critical/High/Medium
findings fixed before commit. Totals across the three waves:
0 Critical ¬∑ 3 High ¬∑ 20 Medium ¬∑ ~30 Low (each disposition is in the
wave's commit message). Run the real Codex gate on the PRs locally.

**Verification-environment note:** the container has no Supabase
credentials, so for in-container UI verification the owner approved
temporarily parking `middleware.ts` while screenshotting (restored before
every commit; never committed). Mock mode drove all views. Two probe
"regressions" were dev-server staleness ‚Äî always clean-restart `next dev`
after large multi-agent rewrites before trusting behavioral probes.
